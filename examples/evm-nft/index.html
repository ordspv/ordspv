An EVM NFT whose image lives on Bitcoin

This ERC-721 token names its image with an ord: URI (a Bitcoin inscription id plus a sha256 integrity pin) instead of a hosted gateway URL. This page pulls that URI out of the token's metadata, resolves it from public Bitcoin data sources, and cryptographically verifies every byte client-side before rendering. The servers are treated as untrusted.
