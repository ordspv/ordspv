Verify a Bitcoin inscription — in this browser

This page fetches inscription 0 from public Bitcoin data sources and cryptographically verifies every byte client-side: transaction hashing, merkle inclusion against a proof-of-work-checked header, and the BIP-341 tapscript commitment. The servers are treated as untrusted. No backend, no build step — view source; the single trust anchor is one pinned block hash you can check against any explorer.

    Verification levels, proof format, and the resolver library live at github.com/ordspv/ordspv (see SPEC-VERIFICATION for L0–L3). Static page; works offline-from-file. Data sources: mempool.space, blockstream.info (failover).
